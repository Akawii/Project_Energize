## Objective
The objective of this project is to develop an intelligent and automated **Bike Storage System** that integrates **IoT**, **Artificial Intelligence**, and **Machine Learning** technologies to enhance the security and accessibility of bicycle parking.  
The system enables users to easily **reserve, access, and pay** for bike lockers through a mobile application, while providing administrators with **real-time monitoring, analytics, and optimization tools** through a connected back office.
